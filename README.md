# FOSS ANDROID APPS

### _Curated list of the best FOSS (free open source software) apps! - Instead of putting multiple apps in a single category, I chose two of the best apps for each after testing and making sure that you get your privacy back._

# CONTRIBUTION

To help out with this project:
-   Find more  **foss** apps and suggest them.
-   Give updates about the repositories of the apps.
-   Fork this repo.
